## Cách chạy Project
### 1. Clone repository từ GitHub: git clone https://github.com/your-repo-url.git
### 2. Di chuyển vào thư mục của dự án: cd myApp
### 3. Cài đặt các dependencies: npm install
### 3. Chạy ứng dụng bằng lệnh: expo start
